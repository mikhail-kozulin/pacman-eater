# Pacman Eater 🟡🐾

Браузерное расширение Chrome: запусти на любом сайте — пакман (с кошачьими ушками) бегает по странице, жрёт интерфейс, оставляет следы из лапок. Бот пытается съесть больше. Серый NPC-кот бродит фоном и какает — наступишь = смерть и респаун.

## Установка для друзей (Chrome / Arc / Brave / Edge — любой Chromium-браузер)

1. **Скачай архив** `pacman-eater.zip` (Михаил пришлёт)
2. **Распакуй** в любую папку (например, `~/Downloads/pacman-eater/`)
3. Открой в браузере `chrome://extensions/`
4. Включи **Developer mode** (правый верхний угол, тумблер)
5. Нажми **Load unpacked** (или «Загрузить распакованное»)
6. Выбери распакованную папку
7. Готово — иконка расширения появится в тулбаре. Если не видно — нажми пазл 🧩 справа от адресной строки и закрепи Pacman Eater булавкой

## Как играть

1. Открой любой сайт (Авито, Яндекс, Пинтерест — что угодно, кроме `chrome://`)
2. Жми иконку расширения 🟡 → «Запустить пакмана»
3. Управление: **WASD** или **стрелки**. **Esc** — выход
4. Цель — съесть на 200 очков раньше бота. Не наступай на 💩
